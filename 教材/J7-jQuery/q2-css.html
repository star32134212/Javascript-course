USING SELECTORS AND EVENTS
jQuery 使用 .CSS() 直接設定 element 的樣式表 只設定單一 declaration 時，直接使用 string 參數，e.g. $('body').css('text-align','center'); 若需同時設定多個 declarations，則可使用 object literal，
e.g.
	$('div').css({
		width:'640px',
		margin:'auto',
		'background-color': 'skyblue',
		color: 'green',
		fontSize: '24px'
	});
